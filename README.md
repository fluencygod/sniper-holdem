# 🎯 저격 홀덤 (Sniper Hold'em)

기존 텍사스 홀덤 규칙에 **'저격(Sniping)'**이라는 고유한 페이즈와 수정된 팟 제한(Pot Limit) 베팅 룰을 접목시킨 커스텀 포커 웹 게임(프로토타입)입니다. Next.js 14, Zustand, Tailwind로 제작되었습니다.

---

## 🚀 다른 환경에서 프로젝트 시작하기 (Getting Started)

저장소를 복제(Clone)하고 로컬 환경에서 바로 실행하는 방법입니다.

### 1단계: 저장소 복제 (Clone)
터미널을 열고 코드를 다운로드받을 폴더로 이동한 뒤 아래 명령어를 입력하세요.
```bash
git clone https://github.com/sy-k-98/sniper-holdem.git
cd sniper-holdem
```

### 2단계: 패키지 설치
이 프로젝트는 `npm`을 사용합니다. 아래 명령어로 의존성 패키지들을 설치하세요.
```bash
npm install
```

### 3단계: 개발 서버 실행
설치가 완료되면 개발 서버를 띄워 게임을 테스트할 수 있습니다.
```bash
npm run dev
```
이후 브라우저에서 `http://localhost:3000` 으로 접속하면 게임 화면을 볼 수 있습니다.

## 📜 게임 룰 소개 (Custom Rules)

### 덱(Deck) 구성
* 1부터 10까지 4가지 무늬(스페이드, 하트, 다이아몬드, 클로버), **총 40장**으로 게임이 진행됩니다.
* 스트레이트 플러시, 플러시 족보 계통은 사용하지 않으며 총 **7가지의 족보**만 인정됩니다. (포카드 > 풀하우스 > 스트레이트 > 트리플 > 투페어 > 원페어 > 하이카드)
* 동점자 발생 시, 1. 족보의 랭크, 2. 핵심 숫자, 3. 개인이 들고 있는 가장 높은 카드, 4. 개인이 들고 있는 나머지 카드의 높낮이 순으로 승패를 가르며 숫자까지 완벽히 동일할 경우 스플릿 팟(Split) 처리됩니다.

### 베팅 룰 (Custom Pot-Limit)
* **베팅 한도**: 현재 살아서 게임 중인(Active) 플레이어들 중 **가장 총 자산(남은칩 + 현재베팅액)이 적은 사람의 금액**까지만 칩을 걸 수 있습니다.

### 게임 페이즈 및 🎯 저격 (Sniping) 시스템
이 게임의 가장 핵심적인 페이즈는 턴 리버 사이인 **SNIPING 페이즈**입니다.

1. **ANTE**: 모두 칩 1개씩 내고 개인 카드 2장을 받습니다.
2. **ROUND 1**: 공용 카드 2장이 공개되고 1차 베팅.
3. **ROUND 2**: 공용 카드 2장(총 4장)이 공개되고 2차 베팅.
4. **SNIPING (저격)**:
   - 각 플레이어는 가장 강력해 보이는 사람의 족보나 뜰 법한 숫자를 저격합니다. (예: `8 스트레이트`, `7 풀하우스`)
5. **SHOWDOWN**: 패 공개 및 승리자 판별.
   - **이때 누군가가 저격한 족보(랭크+핵심숫자)를 가진 플레이어는, 아무리 좋은 패를 가졌더라도 '최하위 족보'로 강등**당합니다!

## 🛠 기술 스택 (Tech Stack)
* **프레임워크**: Next.js 14 (App Router)
* **언어**: TypeScript
* **스타일링**: Tailwind CSS
* **상태 관리**: Zustand (Client-side Game Store)
* **테스트 로직**: Jest (`Judge` 점수 판독 로직 유닛 테스트)
